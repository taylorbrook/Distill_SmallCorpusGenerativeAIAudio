- `class HiFiGANGenerator(nn.Module)`: Takes mel input [B, 128, T], outputs waveform [B, 1, T*512]

- Post-conv: LeakyReLU + Conv1d(channels, 1, 7, padding=3) + tanh
- `remove_weight_norm()` method for inference
- Follow project lazy import pattern: `from __future__ import annotations`

- Use lazy imports consistent with project pattern

**discriminator.py** -- Multi-Period Discriminator (MPD) and Multi-Scale Discriminator (MSD):

`class PeriodDiscriminator(nn.Module)`: Single-period sub-discriminator.
- Reshape input to [B, 1, T//period, period] (pad if needed)
- Stack of 2D convolutions (kernel [5, 1], stride [3, 1]) with LeakyReLU(0.1)
- 4 conv layers with channels [1, 32, 128, 512, 1024] + final conv to 1
- weight_norm on all convs
- Returns (output, list_of_feature_maps)

`class MultiPeriodDiscriminator(nn.Module)`: Wraps 5 PeriodDiscriminators with periods [2, 3, 5, 7, 11] (from config).
- Forward returns (list_of_outputs_real, list_of_outputs_fake, list_of_fmaps_real, list_of_fmaps_fake)

`class ScaleDiscriminator(nn.Module)`: Single-scale sub-discriminator.
- 1D convolution stack: [1, 128, 128, 256, 512, 1024, 1024, 1024, 1] with grouped convs
- weight_norm (or spectral_norm for first scale)
- Returns (output, list_of_feature_maps)

`class MultiScaleDiscriminator(nn.Module)`: 3 ScaleDiscriminators at different scales.
- Scale 0: raw audio, Scale 1: 2x downsampled, Scale 2: 4x downsampled
- Downsampling via AvgPool1d(4, 2, padding=2)
- First scale uses spectral_norm, others use weight_norm
- Forward returns same 4-tuple as MPD

**losses.py** -- Standard HiFi-GAN loss functions (least-squares GAN):

`generator_loss(disc_outputs)`: Sum of (1 - d)^2 for each discriminator output. Returns (total_loss, list_of_per_disc_losses).

`discriminator_loss(disc_real, disc_gen)`: Sum of (1 - dr)^2 + dg^2 for each pair. Returns (total_loss, real_losses, gen_losses).

`feature_loss(fmap_real, fmap_gen)`: L1 distance between intermediate feature maps, multiplied by 2. Returns scalar loss.

All loss functions are pure functions (not nn.Module), accepting lists of tensors.

Update `__init__.py` to export: `MultiPeriodDiscriminator`, `MultiScaleDiscriminator`, `generator_loss`, `discriminator_loss`, `feature_loss`.
